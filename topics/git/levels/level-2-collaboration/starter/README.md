# Starter Project — Git Level 2

This is the starter for the collaboration and conflict resolution exercise.

## Files

- `app.py` — a simple Python greeting script. You and the "teammate" will both edit this
  file's `GREETING` variable, causing a real merge conflict.
- `notes/shared.md` — a notes file that will also be edited on both branches, providing
  a second conflict point.

## How to use

1. Initialize a Git repo in the level-2 folder.
2. Commit these starter files on the `main` branch.
3. Create a feature branch and start editing — the README in the parent folder explains
   the full workflow.
